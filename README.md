# Linux Dotfiles

These won't really be updated too often, as I mainly use MacOS. However, it might be updated ocassionally whenever I decide to boot up Linux. It also only has terminal related things, no tiling window manager (rip Hyprland). More about why in [Inspiration](#inspiration). 

## Inspiration

As I write this (01/20/2026), I am currently in an Intro to Computer Systems class, where we provisioned a [Virtual Machine (VM)](#current-specs-ran-on) using the Google Cloud Platform (GCP). I heard we would be mainly working in the terminal, so I thought it might be a good time to make some dotfiles in case I ever decide to become a Linux user (also if I ever have to do this again, I can set it up fairly quick). With that said, I purely SSH into Linux machines through my Mac, so I currently have no need for a tiling window manager (maybe one day though).

## Current Specs I Use (if you are interested)
- Ubuntu 22.04 "Jammy Jellyfish"
- Intel x86/64 Architecture
- 1 GB Memory
